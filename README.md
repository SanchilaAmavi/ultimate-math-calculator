# Ultimate Math Calculator

A Python-based all-in-one math calculator that can handle:

- Basic arithmetic
- Trigonometry
- Exponentials & logarithms
- Polynomials
- Plotting functions
- Solving equations
- Differentiation
- Integration
- Matrix operations

Everything runs in a command-line interface (CLI).


##  Features

- Math expressions: `2+3*4`, `sin(pi/2)`, `x^2 + 3*x + 2`  
- Plot functions:`plot x^2 + 3*x`, `plot sin(x)`  
- Solve equations: solve x^2 + 3*x + 2`  
- Differentiation:diff x^3 + 5*x`  
- Integration:`integrate x^2 + 3*x`  
- Matrix operations:`matrix [[1,2],[3,4]]`, `matrix [[2,0],[0,2]]`
